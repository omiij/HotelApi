Building Responsive Forms with Flexbox

Additional Resources

I hope you enjoyed Building Responsive Forms with Flexbox! I wanted to share some additional resources on Flexbox that will help you learn more about Flexbox and it's support.

CSS Flexible Box Layout Module

The best way to learn Flexbox is to dive into the specification itself. It doesn't take long to read, and you'll cover all the concepts and syntax rules.

Can I Use Flexbox?

Bookmark caniuse.com's page on Flexbox support and keep track of Flexbox and how it's supported across multiple browsers and devices.

A Complete Guide to Flexbox

CSS Tricks Complete Guide to Flexbox. Excellent quick reference for Flexbox properties and how Flexbox works.

Flexbugs

Phillip Walton's list of known Flexbox bugs with various workarounds.

Why Flexboxes Aren't Good for Page Layout

A leading title to be sure, but Flexbox spec editor Tab Atkins talks about Flexbox, what it's designed to do, and how it will eventually work with other layout modules.

Advanced Responsive Layouts with Flexbox

Morten Rand-Hendriksen's excellent course on creating responsive layouts with Flexbox. Great follow-up to this course.
